Welcome to GitHub Pages.

GArduino is a smart greenhouse that allow you to control some value as Temperature, Air Humidity, Terrain Humidity ecc..

How it's made?

GArduino is made up by Arduino.
Arduino is a programmable electoronic board made in Italy <3 and it's the heart of GArduino.

How GArduino Comunicate?

GArduino comunicate by Internet with the Ethernet Shield and UDP packet.
GArduino Controller allows you in addition to check the value, to Irrigate the greenhouse, switch on/off the Light or activate automatic managemet of the Light.
